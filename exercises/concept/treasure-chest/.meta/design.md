# Design

## Goal

The goal of this exercise is to teach the student the use of `parametric-types` in Julia,.

## Learning objectives

- Know how to use parametric types in Julia for generic programming.

## Out of scope

These will be covered by later concepts:

- Type unions
- Multiple dispatch

## Concepts

- `parametric-types`

## Prerequisites

- `composite-types`
- `nothingness`
